# Virtual Data Room (VDR)

React-based document workspace inside each deal. Folder tree on the left, file table in the middle, AI insights panel on the right.

## Where

- Standalone: [`apps/web/vdr.html`](../../apps/web/vdr.html)
- Embedded in deal page: opens via "Open VDR" header button
- Web-next route: `apps/web-next/src/app/(app)/data-room/`

## Components

| Component | Purpose |
| --- | --- |
| `FolderTree` | Self-referential tree from `Folder.parentId`; drag-drop folder ordering |
| `FiltersBar` | 4 built-in + 7 custom filter chips |
| `FileTable` | Document list with sticky columns |
| `InsightsPanel` | Folder-level AI summary |
| `ConfirmDialog` | Replaces `window.confirm()` (3 variants: danger / warning / info) |
| `Toast` | Replaces `alert()` (4 variants) |

## Smart filters

Built-in:

- Show only PDFs
- Show only Spreadsheets
- AI Warnings (documents flagged by Folder Insights)
- Last 30 Days

Custom presets (added via "+ Custom"):

- Word Docs
- Large Files (> 10 MB)
- Small Files (< 100 KB)
- Last 7 Days / Last 90 Days
- AI Analyzed
- Pending Analysis

## Cross-folder search

Type in the search bar — searches **all folders** in the deal, not just the current one. Result count shown in a blue banner with a clear button.

## Document request

Click "Request document" → enter recipient email + message:

- Resend email is sent with a magic link.
- All deal-team members get an in-app `Notification`.
- An `Activity` row of type `DOCUMENT_REQUESTED` is logged.
- Recipient can upload via the link without an account.

## Folder insights

`FolderInsight` rows are generated by [`folderInsightsGenerator.ts`](../../apps/api/src/services/folderInsightsGenerator.ts):

- `summary` — paragraph
- `completionPercent` — coverage of expected categories
- `redFlags` — string[]
- `missingDocuments` — string[]

Cached, regenerated on demand.

## Gotchas

- Documents without `folderId` vanish from the list. `documents-upload.ts` auto-fills it.
- `overflow-x-auto` clips dropdowns inside the file table — use `flex-wrap` instead.
- Sticky columns must use solid hex `style` backgrounds, not Tailwind classes.
