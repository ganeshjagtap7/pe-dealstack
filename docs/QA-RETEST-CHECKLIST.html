PE OS — New Features Test Checklist

Date: March 11, 2026  |  For: Pushkar  |  New AI features to test

These are brand new AI-powered features we just built. Please test each one and mark Pass / Fail. Screenshot any issues.

1. Contact Enrichment (AI-powered)

Where: Contacts page → click on any contact → look for "AI Enrich" button

#StepExpectedPass?
1.1 Go to Contacts → click on any contact to open detail view Contact detail panel opens
1.2 Look for an "AI Enrich" button in the actions area Button should be visible
1.3 Click "AI Enrich" Should show a loading state, then display enrichment results — company info, LinkedIn data, role details, confidence score
1.4 Check if the enriched data looks reasonable Information should be relevant to the contact (not random data). If confidence is low, it should say "needs review"

2. AI Meeting Prep

Where: Open any deal → click the three-dot menu (⋮) → "Meeting Prep"

#StepExpectedPass?
2.1 Open any deal page (e.g. Luktara) → find the ⋮ menu near the deal header Menu should have a "Meeting Prep" option
2.2 Click "Meeting Prep" Should show a loading state while AI generates the brief
2.3 Wait for the results A modal/panel should appear with a meeting brief containing: talking points, key questions to ask, risk areas, suggested agenda
2.4 Check the content quality Brief should be specific to the deal (mentions company name, financials if available, deal stage). Not generic filler text.
2.5 Close the modal Should close cleanly, back to deal page

3. AI Email Drafter

Where: Open any deal → click the three-dot menu (⋮) → "Draft Email"

#StepExpectedPass?
3.1 Open any deal → ⋮ menu → click "Draft Email" Email drafter modal should open
3.2 Check for template picker — should show different email types Should see 7 templates (e.g. Initial Outreach, Follow Up, NDA Request, LOI, etc.)
3.3 Select a template (e.g. "Initial Outreach") Template should be selected/highlighted
3.4 Check for tone selector Should see 5 tone options (Formal, Friendly, Urgent, etc.)
3.5 Pick a tone → click generate/draft AI should generate an email draft specific to the deal. Should show loading, then the draft text.
3.6 Check if there's a compliance check indicator Should show either "ready for review" (green) or "compliance issues" (warning) status
3.7 Close the modal Should close cleanly

4. Deal Signal Monitor

Where: Dashboard page → look for a signals widget or "Scan Signals" button

#StepExpectedPass?
4.1 Go to Dashboard Look for a Signal Monitor section/widget
4.2 Click "Scan Signals" Should show loading while AI scans your portfolio
4.3 Wait for results Should show signal cards — each with a severity level: Critical (red), Warning (yellow), or Info (blue)
4.4 Check the signal content Signals should reference your actual deals (e.g. "Luktara revenue declining", "Buffer deal stalled"). Not random generic text.

5. Deal AI Chat (Enhanced)

Where: Open any deal → AI Chat panel on the right side

#StepExpectedPass?
5.1 Open a deal that has financials extracted (e.g. Luktara or Pre Flop Wizard) AI Chat panel should be visible on the right
5.2 Ask: "What is the revenue trend for this company?" AI should answer using the deal's actual financial data — mention specific numbers and years
5.3 Ask: "What are the key risks?" Should give deal-specific risk analysis, not generic advice
5.4 Ask: "Compare this deal with Buffer" Should compare the two deals (size, stage, financials if available)
5.5 Upload a document via chat and ask a question about it Should process the document and answer based on its content

6. Portfolio AI Chat

Where: Dashboard or Portfolio page → AI chat/search bar at the top

#StepExpectedPass?
6.1 Go to Dashboard → find the AI search/chat bar at the top Search bar should be visible with placeholder text like "Ask AI anything about your portfolio..."
6.2 Ask: "How many deals are in my pipeline?" Should return the correct count of your deals
6.3 Ask: "Which deals need attention?" Should analyze your portfolio and highlight stalled/at-risk deals
6.4 Ask: "Give me a summary of the Luktara deal" Should pull up Luktara details — stage, size, key metrics

7. Financial Analysis Suite

Where: Open a deal with financials → scroll below Financial Statements section

#StepExpectedPass?
7.1 Open a deal with extracted financials → scroll down past the financial table Should see an "Analysis" section with multiple tabs
7.2 Check for these tabs: Overview, Deep Dive, Cash & Capital, Valuation, Diligence, AI Insights All tabs should be visible and clickable
7.3 Click "Overview" tab Should show QoE score, key ratios, EBITDA bridge chart, overall health indicators
7.4 Click "Deep Dive" tab Should show financial ratios (18 ratios), DuPont analysis, revenue quality breakdown
7.5 Click "Diligence" tab Should show Red Flags section with warnings/errors if any. Cross-document verification results.
7.6 Click "AI Insights" tab Should show AI-generated insights specific to this deal's financials

8. Extract Financials from Data Room (new)

Where: Deal → Data Room tab → file menu

#StepExpectedPass?
8.1 Open a deal → go to Data Room → find a PDF or Excel file File should be visible in the file list
8.2 Click the three-dot menu (⋮) on the file Should see "Extract Financials" option (only for PDF and Excel files)
8.3 Click "Extract Financials" Should show progress, then success message. Financial Statements should populate on the deal page.
